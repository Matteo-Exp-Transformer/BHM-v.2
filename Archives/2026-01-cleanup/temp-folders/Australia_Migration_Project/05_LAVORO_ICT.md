### Struttura consigliata:
```

   - Nome, telefono AU, email, LinkedIn, GitHub/Portfolio

2. PROFESSIONAL SUMMARY (3-4 righe)
   - Chi sei, cosa fai, cosa cerchi

3. TECHNICAL SKILLS

   - Tools: Git, Docker, AWS, etc.

5. PROJECTS (se poca esperienza formale)
   - Nome progetto
   - Tech stack usato
   - Link demo/GitHub
   - Risultati/impatto

6. EDUCATION
   - Degree, Institution, Year

7. CERTIFICATIONS (se rilevanti)
   - AWS Cloud Practitioner, etc.
```
